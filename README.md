# Research Orchestra

Quy trình nghiên cứu có AI hỗ trợ, từ định hướng ban đầu, khám phá ý tưởng và research gap đến đặt câu hỏi, tổng hợp bằng chứng, phản biện và viết paper.

## Chạy local

```bash
./start.sh
./stop.sh
```

Mặc định website chạy tại `http://127.0.0.1:5100`. Có thể thay đổi bằng biến môi trường `HOST` và `PORT`.
